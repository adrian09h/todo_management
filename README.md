# todo_management

## Libraries and tools included:

- Support libraries
- RxJava and RxAndroid
- Retrofit 2
- Dagger 2
- Room
- DataBidning
- RecyclerViews and CardViews
- DiffUtil

- JDK 1.8
- Android SDK
- Latest Android SDK Tools and build tools

This project follows MVVM architecture.
